HTML 表單

來源：html_raw.txt｜表單是 Client 向 Server 送出資料的主要方式

Form 屬性

屬性說明範例
action 送出的目標 URL（Server 端處理程式） /api/register
method get（資料在 URL）或 post（資料在 body） method="post"
enctype 搭配 POST 的資料格式，上傳檔案時需設為 multipart/form-data enctype="multipart/form-data"
target 回應在哪裡開啟（_self / _blank） target="_blank"

GET：資料附在 URL 的 Query String，?name=Ted&age=30，適合搜尋/過濾
POST：資料在 HTTP Request Body 中，適合送出敏感資料（密碼）或大量資料

HTML5 Input 類型總覽

類型用途特別屬性
text一般文字minlength, maxlength, pattern
emailEmail（瀏覽器自動驗證格式）required
password密碼（輸入自動遮蔽）minlength, pattern
number數字min, max, step
date日期選擇器min, max
tel電話（不驗證，需自訂 pattern）pattern
checkbox多選核取方塊checked, value
radio單選按鈕（同 name 為一組）checked, value
file檔案上傳accept, multiple
hidden不顯示但會送出的資料value
range滑桿min, max, step
color顏色選擇器value (#hex)
