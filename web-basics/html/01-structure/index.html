HTML 基礎結構

來源：html_raw.txt｜HTML 是網頁的骨架，把區塊排列出來

基本文件結構

每個 HTML 文件的必要骨架：

<!DOCTYPE html>         <!-- 宣告 HTML5 版本 -->
<html lang="zh-TW">      <!-- 根元素，lang 影響 SEO 和無障礙 -->
  <head>                  <!-- 元資料區（不顯示在頁面上） -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>頁面標題</title>
    <link rel="stylesheet" href="style.css">   <!-- 每個 link 都是一次 HTTP 請求 -->
  </head>
  <body>                  <!-- 可見內容區 -->
    <!-- 頁面內容放這裡 -->
    <script src="app.js"></script>  <!-- script 放 body 底部避免阻塞解析 -->
  </body>
</html>

伺服器回傳 HTML 時，瀏覽器從 <head> 開始逐行解析（L7 應用層）。
遇到 <link> 或 <script> 就發出新的 HTTP 請求取回檔案。
引用的外部檔案越少，載入越快（internet_raw.txt 的效能建議）。

Meta 標籤

Meta 標籤用途
<meta charset="UTF-8">設定字元編碼，確保中文正確顯示
<meta name="viewport"...>讓頁面在行動裝置上「正常」縮放（RWD 必備）
<meta http-equiv="X-UA-Compatible"...>強制 IE 使用最新渲染引擎
<meta name="description"...>SEO：搜尋引擎結果的描述文字
<meta property="og:title"...>Open Graph：社群媒體分享的標題

絕對路徑 vs 相對路徑

以 http://example.com/app/ 為 base URL

寫法解析結果說明
js/main.js http://example.com/app/js/main.js 相對於目前目錄
./js/main.js http://example.com/app/js/main.js 明確指定目前目錄（同上）
../js/main.js http://example.com/js/main.js 上一層目錄
/js/main.js http://example.com/js/main.js 從網域根目錄開始（絕對路徑）
https://cdn.example.com/lib.js 完整 URL 完整絕對路徑（外部資源）

HTML 實體字元（Entities）

某些字元在 HTML 中有特殊含義，需要用實體字元來顯示

實體顯示結果用途
&nbsp; （空格）不換行空格
&lt;<小於符號（避免被解析為標籤）
&gt;>大於符號
&amp;&& 字元本身
&quot;"雙引號
&copy;©版權符號
&reg;®商標符號
&yen;¥日圓符號
&euro;€歐元符號

data-* 自訂屬性

可定義在任何標籤，透過 JavaScript 的 dataset 存取

點擊讀取 data-* 屬性 
<div data-id="42" data-role="admin" data-name="Hsin Hwang">
  ...
</div>

// JavaScript 存取
const el = document.getElementById('user-card');
el.dataset.id   // → "42"
el.dataset.role // → "admin"
el.dataset.name // → "Hsin Hwang"

Canvas 繪圖

Canvas API 透過 JavaScript 在瀏覽器端即時繪製圖形，不需要網路請求

const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');

ctx.fillStyle = '#3182ce';
ctx.fillRect(10, 10, 100, 60);     // 填色矩形

ctx.beginPath();
ctx.arc(200, 60, 40, 0, Math.PI * 2);  // 圓形
ctx.fillStyle = '#68d391';
ctx.fill();

SVG 向量圖形

SVG 是 XML 格式的向量圖，可縮放不失真，可用 CSS/JS 操控

<svg width="300" height="120">
  <!-- cx/cy=圓心, r=半徑, stroke=邊框顏色, fill=填色 -->
  <circle cx="60" cy="60" r="40" stroke="#63b3ed" fill="#1a365d"/>
  <rect x="130" y="20" width="80" height="80" fill="#276749" rx="8"/>
  <text x="240" y="55" fill="#f6e05e">SVG</text>
</svg>
