flex-grow / flex-shrink / flex-basis



  

    
屬性說明預設值

    

      flex-grow
      剩餘空間的分配比例。0 = 不擴張；1 = 平均分配
      0
    

    

      flex-shrink
      空間不足時的縮小比例。0 = 不縮小；1 = 正常縮小
      1
    

    

      flex-basis
      排列前的初始尺寸（主軸方向）
      auto

常見 Flexbox 版型




  
  
① 導覽列（Logo 左，連結右）

      首頁

nav {
  display: flex;
  justify-content: space-between;  /* Logo 左，連結群組右 */
  align-items: center;             /* 垂直置中 */
}

.card-container {
  display: flex;
  flex-wrap: wrap;   /* 超出時換行 */
  gap: 8px;
}
.card {
  flex: 1;           /* 平均分配空間 */
  min-width: 120px;  /* 防止壓縮過小 */
}


  
  
③ 水平+垂直置中（最常用）

  

    
完全置中

  

  
.container {
  display: flex;
  align-items: center;      /* 交叉軸置中（垂直） */
  justify-content: center;  /* 主軸置中（水平） */
}
