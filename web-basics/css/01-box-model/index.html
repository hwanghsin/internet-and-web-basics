box-sizing: content-box vs border-box

content-box（預設）：width 只計算內容，加上 padding/border 後總寬度會增大
border-box（推薦）：width 包含 padding/border，總寬度固定不變

content-box（預設）
width: 160px
+pad:20px+border:4px
= 實際 208px
border-box（推薦）
width: 208px
pad+border 算在內
= 實際 208px ✓ 
/* 最佳實踐：全域套用 border-box */
*, *::before, *::after {
  box-sizing: border-box;
}

Display 屬性比較

值行為可設 width/height？典型使用
block獨佔一整行✅div, p, h1-h6
inline跟著文字流，不換行❌span, a, em, strong
inline-block跟著文字流，但可設尺寸✅按鈕、圖示
flex彈性容器（一維排版）✅導覽列、卡片排列
grid網格容器（二維排版）✅整體版面
none完全消失，不佔空間—隱藏元素
block：獨佔一行 inline-block inline 無寬度 inline-block

visibility:hidden 讓元素不可見但佔空間；display:none 完全移除且不佔空間

CSS 變數 + 函式

/* 宣告 CSS 變數 */
:root {
  --primary: #3182ce;
  --spacing: 16px;
}

/* 使用 CSS 變數 */
.button {
  background: var(--primary);
  padding: var(--spacing);
}

/* calc()：混合單位計算 */
.sidebar { width: calc(100% - 240px); }

/* min() / max() / clamp() */
.text   { font-size: max(14px, 2vw); }         /* 至少 14px */
.image  { width: min(100%, 600px); }            /* 最多 600px */
.title  { font-size: clamp(16px, 4vw, 32px); } /* 16~32px 之間 */
這段文字使用 clamp(14px, 3vw, 28px)，在 14px 和 28px 之間自動縮放
