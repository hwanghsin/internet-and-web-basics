grid-column / grid-row 跨欄跨列



  
Grid 用「線號」定位，從 1 開始。grid-column: 1 / 3 表示從第 1 條線到第 3 條線（佔 2 欄）


  
  

    

      1234
    

    

      

      

      

    

  


  

    
    

      grid-column: 1 / 3
（跨 2 欄）
    

    
Item 2

    
Item 3

    
    

      grid-column: 2 / 4
（跨 2 欄）
    

    
    

      grid-column: 1 / -1
（-1 = 最後一條線，佔整行）
    

  


  
.item-wide    { grid-column: 1 / 3; }   /* 佔第1到第3條線（2欄） */
.item-right   { grid-column: 2 / 4; }   /* 佔第2到第4條線（2欄） */
.item-full    { grid-column: 1 / -1; }  /* -1 = 最後一條線（滿行）*/
.item-tall    { grid-row: 1 / 3; }      /* 跨2列（row方向相同邏輯）*/






常見 Grid 版型




  
  
① Sidebar + Main Content

  

    
Sidebar
固定 160px

    
Main Content
剩餘 1fr

  

  
.layout {
  display: grid;
  grid-template-columns: 160px 1fr;  /* 固定側欄 + 彈性主區 */
  gap: 8px;
}


  
  
② Holy Grail（Header / Sidebar + Main + Aside / Footer）

  

    
Header（grid-column: 1 / -1）

    
Sidebar

    
Main Content

    
Aside

    
Footer（grid-column: 1 / -1）

  

  
.holy-grail {
  display: grid;
  grid-template-columns: 120px 1fr 100px;
  grid-template-rows: auto 1fr auto;
  gap: 6px;
}
.header, .footer { grid-column: 1 / -1; }

  grid-template-columns: repeat(auto-fill, minmax(120px, 1fr));

  /* 不需要 media query，自動響應式！ */
}






Flexbox vs Grid 選擇指南



  

    
場景建議原因

    
導覽列（Logo + 連結）Flexbox一維橫排，justify-content 分配簡單

    
卡片行（一排等寬）Flexboxflex:1 自動等分

    
整體頁面版型Grid需要同時控制列和欄

    
響應式卡片矩陣Gridauto-fill + minmax 免 media query

    
複雜跨欄跨列版型Gridgrid-column / grid-row 精確定位

    
垂直置中單一元素Flexboxalign-items + justify-content 最直覺
