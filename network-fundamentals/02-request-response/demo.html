HTTP Methods

Method語義有 Body？典型使用場景
GET 取得資源否讀取文章、查詢資料
POST 新增資源是送出表單、上傳檔案
PUT 更新資源（整體）是編輯個人資料
DELETE 刪除資源否刪除留言

HTTP 版本演進

版本傳輸層關鍵改進前端影響
HTTP/1.1 TCP 同域最多 6 條連線；隊頭阻塞 需要 CSS Sprite、檔案合併
HTTP/2 TCP 多路復用、Header 壓縮（HPACK）、Server Push 多個小請求不再是效能問題
HTTP/3 UDP（QUIC） 0-RTT 握手、連線遷移、獨立流傳輸 Wi-Fi↔4G 切換時連線不斷
